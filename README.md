# 小牙医乐园 · 儿童牙科候诊安抚中心

一个为候诊区平板和家长手机设计的 Vue3 儿童牙科候诊安抚页面，通过明亮插画、步骤卡片、奖励贴纸和医生头像降低儿童紧张情绪，让孩子觉得可控，让家长知道接下来发生什么。

---

## 原始需求

> 请制作儿童牙科候诊安抚页面，Vue3 页面给候诊区平板和家长手机使用，用明亮插画、步骤卡片、奖励贴纸和医生头像降低儿童紧张。页面展示排队进度、今日检查项目、牙椅步骤、安抚小游戏、术后注意事项和家长确认；护士可以切换叫号状态，医生可以标记补牙、涂氟、窝沟封闭或复诊提醒。这个页面要让孩子觉得可控、让家长知道接下来发生什么，动画和颜色要温和，不能因为可爱元素太多遮住叫号、禁食或复诊信息。

---

## 功能概览

### 🌡️ 三种视图切换
- **👨‍👩‍👧 家长/儿童视图**：排队进度、小游戏、奖励、注意事项、确认
- **📣 护士台视图**：叫号状态一键切换、排队管理
- **🩺 医生工作站**：治疗项目标记、复诊提醒设置

### 📋 核心模块
| 模块 | 说明 |
|------|------|
| 排队进度 | 可视化进度条+小狐狸吉祥物，显示前方等候人数和预计时间 |
| 今日检查项目 | 4大检查项：口腔检查/洁牙/涂氟/窝沟封闭，可标记完成 |
| 牙椅小冒险 | 7步闯关式步骤卡：坐椅子→戴围兜→**拍片**→检查→洁牙→**漱口**→**涂氟**，每步配有专属动画和注意事项 |
| 奖励贴纸墙 | 10枚成就贴纸，分游戏奖励、配合奖励、复诊奖励、特别表扬4类，含里程碑收集系统 |
| 医生问候卡 | 医生头像+亲切问候语，建立信任感 |
| 安抚小游戏 | 3款：🫧戳泡泡 / ⭐接星星 / 🃏记忆翻牌，完成游戏自动获得贴纸奖励 |
| 术后注意事项 | 5大项：进食/清洁/不适/复诊/饮食，警告项特别标注 |
| 家长确认 | 治疗项目汇总+复诊+注意事项+电子签名确认 |

### ✨ 新增功能
| 功能 | 说明 |
|------|------|
| **治疗步骤动画** | 每个步骤配有专属CSS动画，包括拍片X光机扫描、牙刷来回清洁、水流漱口、护盾涂氟保护等 |
| **角色差异化视图** | 家长视图显示详细说明和可展开的注意事项，儿童视图显示友好动画和简单提示 |
| **平滑步骤过渡** | 护士切换步骤时，内容平滑过渡动画，避免孩子以为流程结束 |
| **里程碑收集系统** | 3/5/8/10枚贴纸达成不同成就，进度条可视化显示 |
| **医生鼓励语** | 医生可为获得的贴纸添加个性化鼓励话语 |
| **行为反馈** | 家长端看到的是专业行为反馈而非营销话术 |
| **游戏联动奖励** | 完成小游戏阈值自动触发贴纸奖励，配合弹窗提示 |
| **贴纸来源分类** | 游戏奖励/配合奖励/复诊奖励/特别表扬4类来源标签 |

### 🎨 设计特点
- 温暖奶油色调：珊瑚橙+薄荷绿+柠檬黄+薰衣草紫
- 柔和呼吸动画，避免快速闪烁刺激儿童
- 重要信息（叫号、禁食、复诊）使用高对比度标签突出
- 响应式布局：平板大屏两栏、手机竖屏单栏

## 启动方式

### 前置要求

- **Node.js** >= 18（推荐 20.x LTS）
- **npm** >= 9 或 **pnpm** >= 8
- 现代浏览器（Chrome / Edge / Safari 最新版）

---

### 方式一：本地开发启动（推荐）

#### 1. 安装依赖

```bash
npm install
```

> 如果使用 pnpm：
> ```bash
> pnpm install
> ```

#### 2. 启动开发服务器

```bash
npm run dev
```

启动成功后控制台输出：

```
  VITE v8.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

访问地址：**http://localhost:5173/**

> 如需局域网平板访问，使用 `--host` 参数：
> ```bash
> npm run dev -- --host 0.0.0.0
> ```

#### 3. 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录。

#### 4. 本地预览构建产物

```bash
npm run preview
```

---

### 方式二：Docker 一键启动（推荐用于部署）

#### 前置要求

- **Docker** >= 20.10
- **Docker Compose** >= 2.0（新版 Docker Desktop 已内置）

#### 1. 构建并启动服务

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

#### 2. 访问页面

启动完成后访问：**http://localhost:8080/**

> 如需修改端口，编辑 `docker-compose.yml` 中 `ports` 的第一个数字，例如 `"3000:80"` 即使用 3000 端口。

#### 3. 停止并清理

```bash
docker compose down
```

如需删除挂载卷和数据：

```bash
docker compose down -v
```

#### 4. 仅使用 Dockerfile 单独构建

```bash
# 构建镜像
docker build -t dental-kids-waiting .

# 启动容器
docker run -p 8080:80 --name dental-kids-app dental-kids-waiting
```

## 使用说明

### 护士台操作
1. 顶部切换到 **📣 护士台** 视图
2. 点击右上角 **🔔 切换叫号状态** 按钮循环切换：
   - `排队候诊中` → `请前往诊室` → `治疗中` → `治疗完成`
3. 叫号状态变化时，孩子的号码卡片会高亮提醒
4. **直播使用统计**：查看直播总次数、进行中数量、总观看时长、平均时长
5. **诊室管理**：查看各诊室状态（空闲/准备中/治疗中），可分配患者到诊室
6. **直播记录**：查看所有直播记录，包括开始/结束时间、时长、所属诊室

### 医生工作站操作
1. 顶部切换到 **🩺 医生工作站** 视图
2. 在「术后注意事项」模块中：
   - 点击 **补牙治疗 / 涂氟保护 / 窝沟封闭 / 洁牙抛光** 标记完成
   - 勾选「📅 设置复诊提醒」选择复诊日期
3. 在「今日检查项目」可点击卡片切换完成状态
4. 奖励贴纸墙可点击贴纸手动奖励
5. **治疗状态控制**：
   - 点击「开始治疗」自动关闭诊室直播，保护患者隐私
   - 点击「结束治疗」诊室恢复空闲状态

### 家长 / 儿童视图
1. 查看排队进度，点击小游戏缓解紧张
2. 了解牙椅每一步会发生什么，消除未知恐惧
3. **诊室直播**：
   - 点击「查看诊室直播」可实时查看诊室准备情况
   - 直播仅限本患者家长查看
   - 画面延迟约 2.8 秒，确保隐私安全
   - 医生开始治疗后直播自动关闭，显示「治疗中」占位画面
4. **儿童模式**：
   - 顶部点击「儿童模式 / 家长模式」切换
   - 儿童模式下自动隐藏诊室直播，避免儿童产生焦虑
5. 治疗完成后阅读注意事项（警告项红底突出）
6. 点击「家长确认签字」完成电子确认

## 注意事项

1. **浏览器兼容性**：使用 CSS 变量和新特性，建议使用现代浏览器最新版本
2. **平板适配**：建议屏幕 >= 10 英寸获得最佳两栏体验
3. **动画安全**：所有动画使用 `prefers-reduced-motion` 友好的持续时间（2-5s），避免对光敏感儿童造成不适
4. **重要信息层级**：叫号状态使用脉冲环+黄色边框，禁食注意事项使用红色警告标签，复诊提醒在底部独立显示，确保不被可爱元素遮挡
