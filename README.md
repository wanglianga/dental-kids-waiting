# 小牙医乐园 · 儿童牙科候诊安抚中心

一个为候诊区平板和家长手机设计的 Vue3 儿童牙科候诊安抚页面，通过明亮插画、步骤卡片、奖励贴纸和医生头像降低儿童紧张情绪，让孩子觉得可控，让家长知道接下来发生什么。

---

## 原始需求

> 请制作儿童牙科候诊安抚页面，Vue3 页面给候诊区平板和家长手机使用，用明亮插画、步骤卡片、奖励贴纸和医生头像降低儿童紧张。页面展示排队进度、今日检查项目、牙椅步骤、安抚小游戏、术后注意事项和家长确认；护士可以切换叫号状态，医生可以标记补牙、涂氟、窝沟封闭或复诊提醒。这个页面要让孩子觉得可控、让家长知道接下来发生什么，动画和颜色要温和，不能因为可爱元素太多遮住叫号、禁食或复诊信息。

---

## 功能概览

### 🌡️ 三种视图切换
- **👨‍👩‍👧 家长/儿童视图**：排队进度、小游戏、奖励、注意事项、确认
- **📣 护士台视图**：叫号状态一键切换、排队管理
- **🩺 医生工作站**：治疗项目标记、复诊提醒设置

### 📋 核心模块
| 模块 | 说明 |
|------|------|
| 排队进度 | 可视化进度条+小狐狸吉祥物，显示前方等候人数和预计时间 |
| 今日检查项目 | 4大检查项：口腔检查/洁牙/涂氟/窝沟封闭，可标记完成 |
| 牙椅小冒险 | 6步闯关式步骤卡：坐椅子→戴围兜→照镜子→刷牙→涂漆→漱口 |
| 奖励贴纸墙 | 5枚成就贴纸：勇敢小狮子/闪亮明星/微笑海豚/彩虹勇士/超级牙医 |
| 医生问候卡 | 医生头像+亲切问候语，建立信任感 |
| 安抚小游戏 | 3款：🫧戳泡泡 / ⭐接星星 / 🃏记忆翻牌 |
| 术后注意事项 | 5大项：进食/清洁/不适/复诊/饮食，警告项特别标注 |
| 家长确认 | 治疗项目汇总+复诊+注意事项+电子签名确认 |

### 🎨 设计特点
- 温暖奶油色调：珊瑚橙+薄荷绿+柠檬黄+薰衣草紫
- 柔和呼吸动画，避免快速闪烁刺激儿童
- 重要信息（叫号、禁食、复诊）使用高对比度标签突出
- 响应式布局：平板大屏两栏、手机竖屏单栏

## 启动方式

### 前置要求

- **Node.js** >= 18（推荐 20.x LTS）
- **npm** >= 9 或 **pnpm** >= 8
- 现代浏览器（Chrome / Edge / Safari 最新版）

---

### 方式一：本地开发启动（推荐）

#### 1. 安装依赖

```bash
npm install
```

> 如果使用 pnpm：
> ```bash
> pnpm install
> ```

#### 2. 启动开发服务器

```bash
npm run dev
```

启动成功后控制台输出：

```
  VITE v8.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

访问地址：**http://localhost:5173/**

> 如需局域网平板访问，使用 `--host` 参数：
> ```bash
> npm run dev -- --host 0.0.0.0
> ```

#### 3. 生产构建

```bash
npm run build
```

构建产物在 `dist/` 目录。

#### 4. 本地预览构建产物

```bash
npm run preview
```

---

### 方式二：Docker 一键启动（推荐用于部署）

#### 前置要求

- **Docker** >= 20.10
- **Docker Compose** >= 2.0（新版 Docker Desktop 已内置）

#### 1. 构建并启动服务

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

#### 2. 访问页面

启动完成后访问：**http://localhost:8080/**

> 如需修改端口，编辑 `docker-compose.yml` 中 `ports` 的第一个数字，例如 `"3000:80"` 即使用 3000 端口。

#### 3. 停止并清理

```bash
docker compose down
```

如需删除挂载卷和数据：

```bash
docker compose down -v
```

#### 4. 仅使用 Dockerfile 单独构建

```bash
# 构建镜像
docker build -t dental-kids-waiting .

# 启动容器
docker run -p 8080:80 --name dental-kids-app dental-kids-waiting
```

## 使用说明

### 护士台操作
1. 顶部切换到 **📣 护士台** 视图
2. 点击右上角 **🔔 切换叫号状态** 按钮循环切换：
   - `排队候诊中` → `请前往诊室` → `治疗中` → `治疗完成`
3. 叫号状态变化时，孩子的号码卡片会高亮提醒

### 医生工作站操作
1. 顶部切换到 **🩺 医生工作站** 视图
2. 在「术后注意事项」模块中：
   - 点击 **补牙治疗 / 涂氟保护 / 窝沟封闭 / 洁牙抛光** 标记完成
   - 勾选「📅 设置复诊提醒」选择复诊日期
3. 在「今日检查项目」可点击卡片切换完成状态
4. 奖励贴纸墙可点击贴纸手动奖励

### 家长 / 儿童视图
1. 查看排队进度，点击小游戏缓解紧张
2. 了解牙椅每一步会发生什么，消除未知恐惧
3. 治疗完成后阅读注意事项（警告项红底突出）
4. 点击「家长确认签字」完成电子确认

## 注意事项

1. **浏览器兼容性**：使用 CSS 变量和新特性，建议使用现代浏览器最新版本
2. **平板适配**：建议屏幕 >= 10 英寸获得最佳两栏体验
3. **动画安全**：所有动画使用 `prefers-reduced-motion` 友好的持续时间（2-5s），避免对光敏感儿童造成不适
4. **重要信息层级**：叫号状态使用脉冲环+黄色边框，禁食注意事项使用红色警告标签，复诊提醒在底部独立显示，确保不被可爱元素遮挡
